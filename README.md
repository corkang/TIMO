# 티모(TIMO): 대학생을 위한 Timetable organizer

## 🙌 Overview

티모(TIMO)는 시간표 작성, 공석 알림, 경쟁률 계산 등 시간표와 관련된 다양한 기능을 제공하는 웹 어플리케이션입니다.

## 👥 Team TIMO

[개발] 강신엽, [기획] 김희민, [디자인] 강서영


## 🤩 Main Functionalities

>* 🔍 과목 검색
>* 🗓 시간표 작성
>* 🔥 경쟁률 계산
>* 📖 수강편람 참고
>* 💁🏼‍♀️ 친구와 시간표 공유
>* 📮 공석 알림
>* 🙌 친구와 시간표 비교 (예정)
